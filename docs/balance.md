## Prestige

- Поріг: **1e9** енергії (було 1e10).
- Формула: `baseLog10 = (log10E - 9) / 2 + 1`, тобто:
  - 1e9  → 10 кристалів
  - 1e11 → 100
  - 1e13 → 1 000
- `CRYSTAL_GAIN`: `mult = 1 + 0.25 * level` — кожен тір додає +25%.

## Ядро

- `cost_multiplier`: 3.0 (було 5.0). T1=10, T2=30, T3=90, T4=270, T5=810.
- `effect_value`: 0.15 (було 0.10). Боост = `10^(T * 0.15 * log10(VC))`.
  При 100 VC (log10=2) і T4: `10^(4*0.15*2) = 10^1.2 ≈ 15.8x` глобально.

## Кнопки тюнингу (знати куди крутити)

- **Занадто швидко** → підвищити `PRESTIGE_MIN_LOG10_ENERGY` (9 → 10+).
- **Стіна після gen5** → зменшити `cost_multiplier` для gen5/gen6.
- **Слабкі кристали** → зменшити `PRESTIGE_DIVISOR` (2 → 1.5).
- **Ядро задорого** → зменшити `cost_multiplier` у core (3.0 → 2.5).
